基于克里金法的足球球权热图及价值建模研究——以德甲数据为例

思路概述

基于原本球权热图使用核函数这个基础上进行克里金方向的改进，加入如果在某个区域做某个动作就会有更高价值的新体系定义

克里金法来完成对于权重这方面的权衡

最后的实证分析：找具体比赛和球员，做可视化的分析

第一章 绪论

1.1 研究背景

在现代足球战术体系的演进中，空间控制力（即球队整体的有效控球价值分布）的量化已成为职业球队评估比赛表现、制定战术决策的核心指标之一 。随着光学追踪技术与事件记录系统的普及，足球数据分析正经历从传统的“结果统计”（如控球率、射门数）向高维度的“时空过程分析”跨越。在这一背景下，如何直观且科学地呈现球员或球队在球场上的活动范围与威胁区域，成为了体育分析领域的热点课题。

目前，学术界与工业界生成足球热点图的主流非参数化方法为核密度估计法（Kernel Density Estimation, KDE） 。其核心优势在于计算的高效性与视觉呈现的直观性。通过输入离散的球员触球坐标点，叠加核函数，KDE 能够以极低的时耗生成平滑的概率密度表面，帮助教练组快速建立对球员宏观覆盖范围的认知 。

然而，随着战术分析深度的增加，核密度估计法在球权价值建模中的系统性局限逐渐显现。首先，KDE 本质上是一种基于密度的“计数”逻辑，仅能反映球权发生的“频率”，却无法剥离数据背后的“质量”属性 。在足球比赛中，一次对方禁区前沿的致命直塞，其战术价值远高于己方后场十次毫无压力的横传。KDE 容易导致高频无效触球区域在视觉权重上掩盖低频高价值区域 。其次，KDE 忽略了足球场空间的自相关性趋势，其平滑程度完全取决于人为设定的带宽，容易在数据稀疏区域产生过度平滑或噪声干扰 。更为致命的是，KDE 无法提供反映预测置信度的误差分布图，这使得基于 KDE 生成的战术热图在面对高风险的临场决策时，缺乏严密的统计学支撑 。

针对上述痛点，本研究试图打破传统“频率统计”的桎梏，引入地统计学中专门处理空间自相关性的克里金法，结合德甲联赛的真实事件数据，探索构建一套兼具数学严谨性与战术解释力的足球球权空间价值模型 。

1.2 研究目的与意义

1.2.1 理论意义

本研究推动了足球空间分析从“频率描述”向“统计推断”的实质性跨越。传统基于核密度估计的热图虽然高效，但割裂了足球场上点与点之间内在的空间自相关性。通过引入克里金法，本研究利用变异函数（Variogram）定量表征了足球场上不同区域间的空间依赖特征，为足球地理信息系统分析提供了更严密的统计学基础。更重要的是，克里金法作为一种“最佳线性无偏估计”（BLUE）方法，能够有效地结合实际事件（如传球、射门、防守解围）以及事件发生的动态情景（如位置坐标、是否受到压迫、传球高度等），将离散的球权动作转化为覆盖全场的连续价值曲面 。此外，克里金法独特的预测方差（Kriging Variance）输出功能，为评估空间分析的不确定性提供了数学依据，弥补了现有体育空间分析研究在模型置信度量化方面的理论空白 。

1.2.2 实践意义

在实战应用层面，本研究为职业足球的精准战术复盘与决策提供了科学导航 。相比于 KDE 易受人为带宽设置干扰的缺陷，基于克里金法的平滑热图能更精准地识别出具有统计显著性的“战术走廊”和关键控球区域 。特别是在样本相对稀疏的极端区域，模型生成的误差分布图具有极强的实战预警价值，能够直观标注出数据预测不可信的“情报盲区”，有效降低数据分析师因过度解读噪声而产生的战术失误风险。此外，本研究建立的基于“空间质量”而非“空间频率”的复合评价体系（即球权价值 \(Z\) 模型），能更公平、客观地量化那些在低频区域做出关键战术贡献的球员（如拖后组织核心、空间穿插型前锋等）。这不仅丰富了球队日常的战术诊断工具箱，更为职业俱乐部的引援定价、人才选拔以及对手针对性布阵提供了极具价值的量化依据 。

1.3 国内外研究现状与文献综述

1.3.1 足球时空事件数据集的建立与开放

足球大数据研究的基石在于高质量时空数据集的构建 。Pappalardo等（2019）在其研究中详细介绍了目前足球领域广泛使用的公开数据集，涵盖了数百万个时空匹配的比赛事件数据，这种结构化的数据形式为后续所有关于球权分布与空间建模的研究提供了标准化的底座 。在此基础上，Garrido 等（2022）对比了基于事件数据与追踪数据生成热图的差异，证实了事件数据在捕捉球权转换及特定战术意图方面具有不可替代的业务价值 。

1.3.2 运动空间可视化的演进与 KDE 的局限

空间分析在体育界的前沿探索可追溯至 Goldsberry（2012）提出的“CourtVision”系统 。该研究通过对 NBA 球员投篮位置的空间热点分析，定义了“空间效能”概念，为职业体育的视觉分析奠定了基础 。在足球领域，Link 等（2016）利用核密度估计法（KDE）成功构建了职业足球的实时球权区域分布图，为界定球队势力范围提供了量化标准 。+1然而，Memmert 等（2017）在综述战术分析技术时明确指出，尽管以 KDE为代表的非参数方法应用广泛，但其在处理空间回归与插值时的统计推断能力较弱，难以揭示深层次的空间逻辑关系 。Schultze 与 Wellbrock（2015）进一步提出，现代足球需要更严密的统计与空间分析系统来综合评估球员表现，而不能仅停留在简单的事件密度分布层面 。

1.3.3 克里金法在空间建模中的前沿探索

为了克服 KDE 的局限，部分学者开始关注地统计学方法。Kotzbek 与 Kainz（2015）作为这一领域的先驱，探讨了利用克里金插值处理足球追踪数据并生成“表现表面”的可能性 。Yue 等（2014）通过对传球网络空间关联性的统计分析，证实了足球场上事件分布并非随机，而是存在显著的空间自相关特征，这为克里金法的应用提供了合法性依据 。Erdogan（2015）的跨学科实验也证明，克里金法能够识别被 KDE 忽略的隐藏空间模式，其在热点预测的准确性上显著优于核密度估计 。在足球空间价值建模的前沿，Fernandez 与 Bornn（2018）提出的“开阔空间”量化技术，利用高阶统计建模衡量球员的空间创造能力，这与本研究所倡导的价值建模理念高度契合 。

1.3.4 文献评述

综合上述文献可知，当前足球空间分析正处于从“单纯的可视化展示”向“严密的因果推断与价值评估”过渡的关键时期。尽管高质量的时空数据集已初具规模，但现有的分析工具仍高度依赖统计推断能力较弱的 KDE 方法 。将克里金法系统性地引入足球战术分析，并结合具体赛事数据进行球权价值的重构与验证，目前在学术界仍属于较为前沿且探索空间巨大的领域。

1.4 研究内容与框架

本研究打破传统足球分析中依赖核密度估计的局限，深度探讨球权分布的空间结构及其潜在战术价值 。整体研究框架分为以下五个部分：

  1. 理论基础与指标构建：构建多维度的“球权战术价值（\(Z\)）”指标体系，融合动作基础分、空间向心力权重、防守压迫与动作质量等变量。

  2. 空间相关性验证与变异函数分析：提取德甲赛事的球权事件坐标，通过拟合实验变异函数，定量表征球场上战术事件的空间自相关性，论证克里金模型应用的可行性 。

  3. 克里金插值建模与全景场生成：基于变异函数参数，执行普通克里金插值，生成平滑且连续的“德甲球权价值基准场”，并利用交叉验证（LOOCV）评估模型的绝对误差（RMSE）与预测方差（MSDR）。

  4. 宏观战术实证（单队表现与残差分析）：以特定球队（如拜仁慕尼黑）为例，生成单队战术价值场，并与联盟基准场进行差值计算，生成战术差异图，从而直观呈现球队的战术溢价区与薄弱区 。

  5. 微观球员实证与价值重塑：将模型下沉至个人维度，选取具有典型空间特征的球员，生成个人专属价值热图，探讨模型在评估“空间阅读型”或“低频高价值”球员时的优势。

1.5 研究创新点

  1. 方法论创新：从非参数平滑到参数化统计推断

    本研究创新性地将地质学领域的克里金法系统引入足球战术分析 。通过考虑空间自相关性这一核心变量，模型弥补了传统 KDE 算法在处理离散数据时对空间规律的忽视，提供了具有更强数学解释力的预测表面 。

  2. 评价维度拓展：实现“事件频率”与“战术价值”的深度融合

    突破了现有热图仅展示“球在哪里活动”的表象，本研究自创了多维度的价值加权函数，将球权行为的“质量信息”（如威胁度、受迫状态）转化为空间权重 。生成的模型不再是一张活动轨迹图，而是动态的“空间潜在收益图” 。

  3. 决策安全性创新：引入数据置信度量化机制

    利用克里金法输出预测方差的特性，本研究在战术分析中引入了“置信度”与“不确定性”概念 。这使得分析师能够清晰识别由样本不足引发的统计伪影，为现代足球的高风险战术决策提供了可量化的安全边界。

第三章 概念界定与理论模型介绍

3.1 传统空间分布估计模型

3.1.1 核密度估计法 (Kernel Density Estimation, KDE)

核密度估计法是目前足球数据分析中生成“热点图”最广泛使用的非参数化空间统计方法。其基本原理是在空间中每一个离散的事件发生点（如球员触球坐标）上放置一个平滑的核函数，通过将所有点的核函数进行叠加，计算出整个二维空间内的概率密度表面。

对于二维足球场空间，其核密度估计的数学表达式通常为：

\[\hat{f}(x,y) = \frac{1}{n h_x h_y} \sum_{i=1}^{n} K\left(\frac{x - x_i}{h_x}, \frac{y - y_i}{h_y}\right)\]

其中，\(n\) 为样本点总数，\(h_x, h_y\) 为人为设定的平滑带宽，\(K\) 为核函数（通常选用高斯核）。

概念局限性：KDE 本质上刻画的是“事件发生的频率密度”。在足球场景中，它只能回答“球员在哪里触球最多”，却无法回答“哪里的触球最具战术威胁”。同时，其平滑结果极度依赖主观设定的带宽 \(h\)，且假设空间中的独立事件彼此无关，忽略了足球运动中天然存在的空间自相关性。

3.2 空间统计学与克里金插值模型

为了克服 KDE 的局限，本研究引入地统计学模型。地统计学的核心逻辑在于，它超越了传统非参数平滑算法的机械刻画，深刻承认了空间分布数据所独具的“结构性”与“随机性”双重特征。具体而言，“结构性”是指足球场上的战术价值并非随机散布，而是受到距离衰减规律和战术阵型的约束，呈现出强烈的空间依赖与连续演变趋势；而“随机性”则表征了在复杂的体育竞技环境中，由于测量误差、球员个体决策偏差以及不可预测的球权物理反弹所造成的局部空间异质性。通过统筹这两种特性，地统计学模型能够以更为严密的数学逻辑，还原出高度契合真实比赛环境的空间价值场。

3.2.1 空间自相关与战术溢出效应

空间自相关是地统计学区别于经典统计学的核心概念。经典统计学通常假设样本之间是独立同分布的，但在处理真实世界的空间数据时，这一假设往往失效。1970年，瓦尔多·托布勒（Waldo Tobler）提出了著名的“托布勒地理学第一定律”：“任何事物都是与其他事物相关的，但相近的事物关联更紧密。”这一定律奠定了空间自相关理论的基石，即空间位置上相近的变量取值之间存在着内在的统计学依赖关系。

在高度复杂且动态变化的足球场上，这一地理学定律展现出了极强的战术适用性。足球比赛本质上是对空间的争夺与利用，球场上的战术事件绝非孤立发生，而是具有显著的“空间溢出效应”。某一区域的高质量控球，往往会跨越物理距离，对其相邻甚至更远的区域产生战术辐射和防守牵制。

具体而言，这种空间自相关性在足球战术中主要体现在以下两个维度：

  1. 进攻威胁的辐射性（正向空间自相关）：当进攻方在诸如肋部区域或大禁区前沿完成一次极具威胁的持球推进时，不仅该坐标点本身的战术价值剧增，其相邻的禁区中路或底线传中区的潜在威胁值也会随之呈现指数级上升。这种高价值区域在空间上的聚集特征，在统计学上表现为显著的正向空间自相关。
  2. 防守重心的拉扯（防守引力效应）：某一边路的高频触球会形成防守引力，迫使防守方阵型整体倾斜。这种局部空间的拥挤，会在球场的另一侧（弱侧）暴露出高价值的开阔空间。这种由一侧的行动引起另一侧价值变化的联动关系，进一步证明了球场空间是一个不可割裂的有机整体。

传统基于核密度估计或简单网格划分的统计方法，其致命弱点正是将球场上每一米、每一次触球都视为相互独立的离散事件，粗暴地剥离了上述的空间依赖关系。因此，本研究引入空间自相关理论，目的正是为了在数学层面上还原足球场上点与点之间隐蔽的战术联系。只有承认并量化了这种空间自相关性，我们才能顺理成章地使用变异函数去测量这种战术辐射的有效半径，进而为克里金插值提供科学的推断依据。

3.2.2 变异函数

变异函数（或半变异函数）是量化空间自相关性的核心数学工具。它描述了空间变量随着距离的增加，其空间依赖性是如何减弱的。 设 \(Z(x_{\alpha})\) 为球场上位置 \(x_{\alpha}\) 处的球权战术价值，定义其空间实验变异函数 \(\gamma(h)\) 为：

\[\gamma(h) = \frac{1}{2N(h)} \sum_{i=1}^{N(h)} [Z(x_i) - Z(x_i + h)]^2\]

其中，\(h\) 为空间滞后距离（即两点间的距离），\(N(h)\) 为距离为 \(h\) 的样本点对数量。

本研究采用指数模型对实验变异函数进行理论拟合。其拟合结果包含三个极其重要的地统计学概念：

  1. 块金值 (Nugget)：当距离 \(h\) 趋近于 0 时，变异函数不为 0 的常数项，代表由测量误差或微观随机性（如足球场上不可预测的球权反弹）造成的空间异质性。
  2. 基台值 (Sill)：随着距离的增加，变异函数达到平稳状态的极限值，代表变量在全场的总空间变异程度。
  3. 变程 (Range)：变异函数达到基台值时的距离。在足球语境中，变程具有极其重要的战术意义，它代表了球员战术动作在球场上的“最大有效辐射半径”。超过此距离，事件之间即被视为统计独立。

3.2.3 普通克里金插值法及其数学推导

普通克里金法是地统计学中应用最为广泛的空间局部插值方法。与传统基于纯几何距离的反距离权重法（IDW）不同，普通克里金法不仅仅考虑已知点与预测点之间的距离，更通过变异函数将已知点之间的空间拓扑关系（如数据的聚集与遮蔽效应）纳入计算。它被称为“最佳线性无偏估计”（BLUE），其核心在于在满足无偏性的前提下，使得预测误差的方差最小。

1. 理论前提：内蕴平稳性假设

普通克里金法的应用建立在内蕴平稳性假设之上。在足球场空间分布的语境下，该假设意味着：

  1. 期望的恒定性：在局部的搜索邻域内，球权战术价值 \(Z(x)\) 的数学期望是一个常数，虽未知但固定，即 \(E[Z(x)] = c\)（常数）。
  2. 增量的平稳性：球场上任意两点 \(x_i\) 和 \(x_i+h\) 处的球权价值之差的方差，仅与两点间的空间向量 \(h\) 有关，而与它们在球场上的绝对位置无关。这一特性正是前文构建变异函数 \(\gamma(h)\) 的理论基础。

2. 线性无偏估计的构建

对于球场上任意未发生实际触球的空白网格点 \(x_0\)，可以利用其周围邻域内的 \(n\) 个已知有效样本点 \(Z(x_1), Z(x_2), \dots, Z(x_n)\)，来估算该点的战术价值 \(Z^*(x_0)\)。普通克里金法采用已知样本值的线性组合来进行预测：

\[Z^*(x_0) = \sum_{i=1}^{n} \lambda_i Z(x_i)\]

其中，\(\lambda_i\) 为赋予第 \(i\) 个已知样本点的克里金权重系数。

为了保证该估计是无偏的，即估计值的期望必须等于真实值的期望，必须满足以下条件：

\[E[Z^*(x_0) - Z(x_0)] = 0\]

代入内蕴平稳性假设 \(E[Z(x)] = c\)，可推导出权重系数的无偏约束条件：

\[\sum_{i=1}^{n} \lambda_i = 1\]

3. 估计方差最小化与拉格朗日乘数法

克里金法被称为“最佳（Best）”估计，是因为它要在所有满足 \(\sum \lambda_i = 1\) 的线性组合中，寻找到一组最优的权重 \(\lambda_i\)，使得预测值与真实值之间的估计方差最小。 估计方差 \(\sigma_E^2\) 可表示为：

\[\sigma_E^2 = Var[Z^*(x_0) - Z(x_0)]\]

将该方差表达式利用前文定义的变异函数 \(\gamma(h)\) 展开，可以将最小化方差的优化目标转化为变异函数的表达式：

\[\sigma_E^2 = 2 \sum_{i=1}^{n} \lambda_i \gamma(x_i, x_0) - \sum_{i=1}^{n} \sum_{j=1}^{n} \lambda_i \lambda_j \gamma(x_i, x_j)\]

这是一个带有等式约束的非线性极值问题。为了求解使得 \(\sigma_E^2\) 最小的 \(\lambda_i\)，这里引入拉格朗日乘数法。构造拉格朗日目标函数 \(L\)：

\[L(\lambda_i, \mu) = \sigma_E^2 + 2\mu \left( \sum_{i=1}^{n} \lambda_i - 1 \right)\]

其中，\(\mu\) 为拉格朗日乘数。

4. 普通克里金方程组

为了求得极小值，将目标函数 \(L\) 分别对所有的权重 \(\lambda_i\) 和拉格朗日乘数 \(\mu\) 求偏导数，并令偏导数等于零：

\[\frac{\partial L}{\partial \lambda_i} = 0 \quad (i = 1, 2, \dots, n)\]

\[\frac{\partial L}{\partial \mu} = 0\]

经过化简，得到普通克里金方程组：

\[\begin{cases} \sum_{j=1}^{n} \lambda_j \gamma(x_i, x_j) + \mu = \gamma(x_i, x_0) & (i = 1, 2, \dots, n) \\ \sum_{i=1}^{n} \lambda_i = 1 & \end{cases}\]

在实际的足球空间建模计算中（如本研究中调用的 gstat 核心算法），计算机会将上述方程组转化为矩阵运算形式：

\[\begin{bmatrix} \gamma(x_1, x_1) & \cdots & \gamma(x_1, x_n) & 1 \\ \vdots & \ddots & \vdots & \vdots \\ \gamma(x_n, x_1) & \cdots & \gamma(x_n, x_n) & 1 \\ 1 & \cdots & 1 & 0 \end{bmatrix} \begin{bmatrix} \lambda_1 \\ \vdots \\ \lambda_n \\ \mu \end{bmatrix} = \begin{bmatrix} \gamma(x_1, x_0) \\ \vdots \\ \gamma(x_n, x_0) \\ 1 \end{bmatrix}\]

通过对左侧的半方差矩阵求逆，即可解出每一个已知触球点对未知网格点的最优权重 \(\lambda_i\)。

5. 克里金方差与数据置信度量化

将求解出的最优权重 \(\lambda_i\) 代回估计方差公式中，即可求得克里金预测方差（\(\sigma_K^2\)）：

\[\sigma_K^2 = \sum_{i=1}^{n} \lambda_i \gamma(x_i, x_0) + \mu\]

在足球战术分析中，克里金方差具有颠覆性的实际意义。传统基于核密度（KDE）的热点图只能给出预测的绝对值，却无法告诉你这个预测结果的偏移可能性。而克里金模型输出的 \(\sigma_K^2\) 完美弥补了这一空白：当 \(\sigma_K^2\) 较小时，说明该区域附近的真实触球样本点丰富且规律性强，战术价值的预测可信度极高；反之，若某一边角区域 \(\sigma_K^2\) 畸高，模型则会在数学上发出预警——该区域的热度预测可能因样本稀疏而存在较大误差。这种对预测不确定性的量化，为足球教练组的实战排兵布阵确立了科学的风险评估边界。

3.3 足球球权战术价值评价体系（Z 模型）的构建

本研究的根本创新在于将原本离散、无属性的坐标点，转化为具有明确战术指向的“价值点”。为此，本研究摒弃了单一维度的评价方式，构建了多维度的“球权战术价值（\(Z\)）”乘法模型。在正式确立模型公式之前，本节将首先从战术理论与客观数据双重维度，论证核心特征选取的科学性。

3.3.1 模型特征选择的理论与数据依据

本研究的 Z 模型包含四个核心维度：动作类型、空间位置、防守压迫与传球高度。其选取依据不仅立足于前沿的运动科学理论，更得到了德甲 15/16 赛季全样本数据的客观支撑。

1. 动作类型维度 (\(S_{base}\)) 的选取依据

理论支持：基于马尔可夫链的“动作状态转换”理论 在现代足球量化分析中，Decroos 等人 (2019) 提出的 VAEP (Valuing Actions by Estimating Probabilities) 框架确立了基于事件驱动（Event-driven）的动作评价范式。该理论将足球比赛视为一系列离散动作的状态转移过程，证明了传球、射门、防守解围等不同行为对预期进球（xG）的贡献度存在显著的层级差异。本研究设定的基础动作分（\(S_{base}\)）正是基于这一状态转换理论，旨在对不同技术动作赋予其天然的期望收益权重。

数据支持：定位球的高转化率溢价 传统基于欧氏距离的空间模型往往会严重低估角球的价值，因为其发生地位于球场四个角落（远离球门中心）。然而，对德甲数据的实证检验推翻了这一直觉：

数据结果显示，角球直接形成射门助攻（Shot Assist）的概率远高于运动战常规传球。这证明了定位球具有跨越空间距离直接威胁球门的能力，为后续模型中赋予定位球极高的基础分与空间豁免权提供了坚实依据。

2. 空间位置维度 (\(W_{space}\)) 的选取依据

理论支持：空间效能与“Zone 14”理论 足球场上的空间效能并非均匀分布。Horn 等人 (2002) 提出的‘Zone 14（即大禁区前沿中路区域）’理论指出，在该区域的有效触球与最终进球转化率呈高度正相关。同时，Fernandez 与 Bornn (2018) 的开阔空间模型也证实了进攻向心力对防守阵型的破坏作用。基于此，本模型引入空间坐标，构建了‘横向向心与纵向推进’的双维空间权重。

数据验证：射门事件的二维核密度分布 为验证上述理论，本研究提取全赛季射门数据并绘制了空间密度分布图：

如核密度图所示，绝对得分机会高度向心聚拢于对方禁区中路（\(X > 100, Y \in [30, 50]\)）。这在统计学上完美印证了“向心推进”的战术溢价规律。

3. 防守压迫维度 (\(C_{pressure}\)) 的选取依据

理论支持：生态动力学中的“约束限制”理论 运动生态动力学（Ecological Dynamics）认为，运动员的决策和动作价值不能脱离环境约束（Constraints）。传统的数据统计存在“数据欺骗性”——后卫在无压力下的横传与中场在被包夹下的出球，其战术难度截然不同。StatsBomb 数据集率先引入的“受迫（Under Pressure）”标签，使得剥离“无效控球”成为可能。

4. 传球高度维度 (\(Q_{quality}\)) 的选取依据

理论支持：瓜迪奥拉的“地面控制”战术哲学 本研究的实证对象为 15/16 赛季的德甲联赛（瓜迪奥拉执教拜仁慕尼黑的巅峰时期）。现代传控足球的战术哲学极度强调对球权的绝对控制，而地面传球（Ground Pass）相比于高空球（High Pass）拥有更高的球权保留率和更低的失误方差。因此，引入传球高度作为控球质量修正参数（\(Q_{quality}\)），是对赛事时代背景与主流战术理念的客观还原。

3.3.2 Z 模型的多维特征乘法构建

在确立上述特征维度的合理性后，本研究采用多维特征乘法模型构建最终的球权战术价值指标 \(Z\)。公式定义如下：

\[Z = S_{base} \times W_{space}(x, y) \times C_{pressure} \times Q_{quality}\]

各子模块的具体数学设定如下：

1. 基础动作分 (\(S_{base}\))

模型依据战术影响程度，对不同类型的技术动作进行固有威胁等级赋值（详见表2-1）。特别设立了针对失误行为的负分机制。

表2-1 基础动作战术价值等级赋值表

动作类型 (Event Type) \(S_{base}\) 分值 战术含义
射门 / 关键扑救 (Shot / Save) 10.0 直接改变比分或阻止得分
助攻 / 关键传球 (Key Pass) 8.0 直接创造射门机会
门线解围 / 封堵 (Clearance / Block) 8.0 破坏绝对得分机会
角球 (Corner) 7.0 [特例]高价值定位球机会
任意球 / 拦截 / 抢断 / 过人 6.0 夺回球权或破坏防守结构
对抗 / 施压 / 护球 4.0 有效的战术物理对抗
普通传球 / 推进 1.0 维持球权周转
普通动作 (其他) 1.0 常规触球
失误 (Error/Loss) -3.0 丢失球权或战术执行失败

2. 多维空间权重 (\(W_{space}\))

该模块综合了纵向推进 (\(X\)) 和横向向心 (\(Y\)) 的位置价值。首先定义 Y 轴向心度函数 \(C_{y}(y)\)，其中 \(y \in [0, 80]\)：

\[C_{y}(y) = 1 - \frac{|y - 40|}{40}\]

该函数确保了中轴线 (\(y=40\)) 取得最高分 1.0，向两侧边线线性递减至 0。

随后，根据动作性质将 \(W_{space}\) 划分为五种应用情境：

  • 情形 A：角球 (Corner) —— [特权豁免]：\(W_{space} = 2.5\)（赋予理论最高权重，无视其边路发球的位置劣势）。
  • 情形 B：任意球 (Free Kick) —— [补偿豁免]：\(W_{space} = 2.0\)。
  • 情形 C：防守动作 —— 目标为保护己方球门：

\[W_{space}(x, y) = 1 + \underbrace{\frac{120 - x}{120}}_{\text{后场保护}} + \underbrace{0.5 \times C_{y}(y)}_{\text{中路防守}}\]

  • 情形 D：进攻动作 —— 目标为攻击对方球门：

\[W_{space}(x, y) = 1 + \underbrace{\frac{x}{120}}_{\text{前场推进}} + \underbrace{0.5 \times C_{y}(y)}_{\text{中路渗透}}\]

  • 情形 E：失误 (Errors) —— \(W_{space} = 0.5\)（进行空间降维，防止在对方危险区域失误反而获得高分变异）。

3. 压力情景系数 (\(C_{pressure}\))

基于前文验证的对抗高风险属性，设定二元分段函数奖励抗压迫操作：

\[C_{pressure} = \begin{cases} 1.5, & \text{if Under Pressure = TRUE} \\ 1.0, & \text{if Under Pressure = FALSE} \end{cases}\]

4. 控球质量修正 (\(Q_{quality}\))

针对传球事件，以地面控制作为最优解进行质量修正：

\[Q_{quality} = \begin{cases} 0.8, & \text{if High Pass (高空球)} \\ 0.9, & \text{if Low Pass (半高球)} \\ 1.0, & \text{if Ground Pass (地面球) or Non-Pass Actions} \end{cases}\]

3.3.3 模型算法的程序实现与逻辑校验

基于上述严密的数学设定，本研究利用 R 语言中的 dplyr 数据操作框架完成了 Z 模型的程序化落地。核心代码通过 case_when 嵌套逻辑，精确计算了全赛季每一条事件记录的合成最终价值 \(Z\)：

final_model_data <- full_season_cleaned %>%
  mutate(
    # 1. 识别并归类定位球等特殊事件
    custom_type = case_when(
      type.name == "Pass" & pass.type.name == "Corner" ~ "Corner",
      type.name == "Pass" & pass.type.name == "Free Kick" ~ "Free Kick",
      TRUE ~ type.name
    ),
    
    # 2. 计算向心度与基础分
    centrality_score = 1 - (abs(location.y - 40) / 40),
    s_base = case_when(
      custom_type %in% c("Shot", "Goal Keeper") ~ 10,
      custom_type == "Corner" ~ 7, 
      custom_type == "Free Kick" ~ 6,
      type.name == "Pass" & pass.shot_assist == TRUE ~ 8,
      custom_type %in% c("Clearance", "Block") ~ 8,
      custom_type %in% c("Interception", "Ball Recovery", "Foul Won", "Dribble") ~ 6,
      custom_type %in% c("Duel", "Pressure", "Shield") ~ 4,
      custom_type %in% error_types ~ -3,
      TRUE ~ 1
    ),

    # 3. 计算多维空间权重
    w_space = case_when(
      custom_type == "Corner" ~ 2.5,
      custom_type == "Free Kick" ~ 2.0,
      custom_type %in% error_types ~ 0.5,
      custom_type %in% def_types ~ 1 + ((120 - location.x) / 120) + (0.5 * centrality_score),
      TRUE ~ 1 + (location.x / 120) + (0.5 * centrality_score)
    ),
    
    # 4. 压力系数与质量系数修正
    c_pressure = ifelse(under_pressure == TRUE, 1.5, 1.0),
    q_height = case_when(
      type.name == "Pass" & pass.height.name == "High Pass" ~ 0.8,
      type.name == "Pass" & pass.height.name == "Low Pass" ~ 0.9,
      TRUE ~ 1.0
    ),
    
    # 5. 合成最终战术价值 Z
    value_z = s_base * w_space * c_pressure * q_height
  ) %>%
  filter(!is.na(value_z))

为确保特征赋值与预期设计一致，模型执行了分组均值校验（Group Means Check）：

空间效能与“Zone 14”理论 (\(W_{space}\))

为什么选“空间坐标 X, Y”？

足球场上的空间效能并非均匀分布。Horn 等人 (2002) 提出的‘Zone 14（即大禁区前沿中路区域）’理论指出，在该区域的有效触球与最终进球转化率呈高度正相关。同时，Fernandez 与 Bornn (2018) 的开阔空间模型也证实了进攻向心力对防守阵型的破坏作用 。基于此，本模型引入空间坐标，构建了‘横向向心与纵向推进’的双维空间权重（\(W_{space}\)）。

生态动力学中的“约束限制”理论 (\(C_{pressure}\))

为什么选“受压迫状态”？

运动生态动力学（Ecological Dynamics）认为，运动员的决策和动作价值不能脱离环境约束（Constraints）。传统的数据统计（如单纯的传球成功率）存在“数据欺骗性”——后卫在无压力下的横传与中场在被包夹下的出球，其战术价值截然不同。StatsBomb 数据集是业内率先引入“受迫（Under Pressure）”标签的系统，本模型利用这一标签构建压力系数（\(C_{pressure}\)），是为了剥离“无效控球”，重奖在极端防守约束下完成的高难度战术动作。

理论支持四：瓜迪奥拉的“地面控制”战术哲学 (\(Q_{quality}\))

为什么选“传球高度”？

本研究的实证对象为 15/16 赛季的德甲联赛（瓜迪奥拉执教拜仁慕尼黑时期）。现代传控足球的战术哲学极度强调对球权的绝对控制，而地面传球（Ground Pass）相比于高空球（High Pass）拥有更高的球权保留率和更低的失误方差。因此，引入传球高度作为控球质量修正参数（\(Q_{quality}\)），是对赛事时代背景与主流战术理念的客观反映。

公式： \[Z = S_{base} \times W_{space}(x, y) \times C_{pressure} \times Q_{quality}\]

其中：

\(S_{base}\)：基础动作分 (Base Action Score)

\(W_{space}\)：多维空间权重 (Multi-dimensional Spatial Weight)

\(C_{pressure}\)：压力情景系数 (Pressure Coefficient)

\(Q_{quality}\)：控球质量修正 (Quality/Height Modifier)

反映不同战术动作的固有威胁或价值等级赋值：

动作类型 (Event Type) \(S_{base}\) 分值 战术含义
射门 / 关键扑救 (Shot / Save) 10.0 直接改变比分或阻止得分
助攻 / 关键传球 (Key Pass) 8.0 直接创造造射门机会
门线解围 / 封堵 (Clearance / Block) 8.0 破坏绝对得分机会
角球 (Corner) 7.0 [特例]高价值定位球机会
任意球 / 拦截 / 抢断 / 过人 6.0 夺回球权或破坏防守结构
构对抗 / 施压 / 护球 4.0 有效的战术对抗
普通传球 / 推进 1.0 维持球权周转
普通动作 (其他) 1.0 常规触球
失误 (Error/Loss) -3.0 丢失球权或战术失败

第二部份：空间因素

多维空间权重 (\(W_{space}\))这是模型的核心，综合了纵向推进 (X) 和 横向向心 (Y) 的价值，并对定位球进行特殊豁免。

首先定义 Y轴向心度函数 \(C_{y}(y)\)，其中 \(y \in [0, 80]\)：

\[C_{y}(y) = 1 - \frac{|y - 40|}{40}\]

(解释：在中轴线 \(y=40\) 处取最大值 1.0，在边线处取 0)

根据动作性质，\(W_{space}\) 分为四种情况：

情形 A：角球 (Corner) —— [特权豁免]

\[W_{space} = 2.5\]

(直接赋予理论最高权重，无视边路位置劣势)

情形 B：任意球 (Free Kick) —— [补偿豁免]\[W_{space} = 2.0\]

情形 C：防守动作 (Defensive Actions) —— 保护己方球门

\[W_{space}(x, y) = 1 + \underbrace{\frac{120 - x}{120}}_{\text{后场保护}} + \underbrace{0.5 \times C_{y}(y)}_{\text{中路防守}}\]

(解释：越靠近己方底线 \(x=0\) 分越高，越靠近中路分越高)

情形 D：进攻动作 (Offensive Actions) —— 攻击对方球门

\[W_{space}(x, y) = 1 + \underbrace{\frac{x}{120}}_{\text{前场推进}} + \underbrace{0.5 \times C_{y}(y)}_{\text{中路渗透}}\]

(解释：越靠近对方底线 \(x=120\) 分越高，越靠近中路分越高)

情形 E：失误 (Errors)

\[W_{space} = 0.5\]

(不进行空间放大，防止在对方禁区失误反而获得高分)

第三部份：压迫情景

压力情景系数 (\(C_{pressure}\))奖励在高强度对抗下的成功操作：

\[C_{pressure} = \begin{cases} 1.5, & \text{if Under Pressure = TRUE} \\ 1.0, & \text{if Under Pressure = FALSE} \end{cases}\]

第四部份：传球高度

控球质量修正 (\(Q_{quality}\))针对传球动作，基于球的飞行高度进行修正（奖励地面控制）：

\[Q_{quality} = \begin{cases} 0.8, & \text{if High Pass (高空球)} \\ 0.9, & \text{if Low Pass (半高球)} \\ 1.0, & \text{if Ground Pass (地面球) or Non-Pass Actions} \end{cases}\]

final_model_data <- full_season_cleaned %>%
  mutate(
    # 特殊情况：角球和任意球
    custom_type = case_when(
      # 优先识别定位球
      type.name == "Pass" & pass.type.name == "Corner" ~ "Corner",
      type.name == "Pass" & pass.type.name == "Free Kick" ~ "Free Kick",
      
      # 其他保持原样
      TRUE ~ type.name
    ),
    
    # Y 轴中心度
    # 距离中轴线(40)越近，分数越接近 1；在边线则为 0
    centrality_score = 1 - (abs(location.y - 40) / 40),
    
    # 基础分 S_base (基于custom_type)
    s_base = case_when(
      # 射门/扑救
      custom_type == "Shot" ~ 10,
      custom_type == "Goal Keeper" ~ 10,
      
      # 关键传球/定位球
      # 角球
      custom_type == "Corner" ~ 7, 
      # 任意球
      custom_type == "Free Kick" ~ 6,
      # 运动战关键传球
      type.name == "Pass" & pass.shot_assist == TRUE ~ 8,
      
      # 关键防守
      custom_type %in% c("Clearance", "Block") ~ 8,
      custom_type %in% c("Interception", "Ball Recovery", "Foul Won") ~ 6,
      
      # 技术动作
      custom_type == "Dribble" & dribble.outcome.name == "Complete" ~ 6,
      
      # 普通动作
      custom_type %in% c("Duel", "Pressure", "Shield") ~ 4,
      custom_type %in% c("Pass", "Carry", "Ball Receipt*") ~ 1,
      
      # 失误
      custom_type %in% error_types ~ -3,
      
      TRUE ~ 1
    ),

    # 空间权重 W_space (X推进 + Y向心 + 特权豁免)
    w_space = case_when(
      
      # 角球：直接给满级权重 (2.5)，无视它在边路
      custom_type == "Corner" ~ 2.5,
      
      # 任意球：给予较高的保底权重
      custom_type == "Free Kick" ~ 2.0,
      
      # 失误：不放大
      custom_type %in% error_types ~ 0.5,
      
      # 防守：目标是 x=0 (保底线) + y=40 (守中路)
      # 1 + 纵向分 + (0.5 * 横向分)
      custom_type %in% def_types ~ 1 + ((120 - location.x) / 120) + (0.5 * centrality_score),
      
      # 进攻组：目标是 x=120 (攻底线) + y=40 (走中路)
      TRUE ~ 1 + (location.x / 120) + (0.5 * centrality_score)
    ),
    
    # 压力与质量
    c_pressure = ifelse(under_pressure == TRUE, 1.5, 1.0),
    
    q_height = case_when(
      type.name == "Pass" & pass.height.name == "High Pass" ~ 0.8,
      type.name == "Pass" & pass.height.name == "Low Pass" ~ 0.9,
      TRUE ~ 1.0
    ),
    
    # 合成最终价值 Z
    value_z = s_base * w_space * c_pressure * q_height
  ) %>%
  filter(!is.na(value_z))
